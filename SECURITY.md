# Security Policy

This repository is a local research toolkit. It does not run a hosted service
or handle credentials by default.

Please do not open public issues for suspected vulnerabilities. Instead,
contact the repository maintainer privately with:

- affected version or commit,
- reproduction steps,
- impact,
- suggested mitigation if known.

Do not include private API keys, brokerage credentials, wallet seeds, or
personal financial data in reports, fixtures, issues, or pull requests.
